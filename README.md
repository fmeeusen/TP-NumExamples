# TP-NumExamples
A research project we did for our BSc level Transport Phenomena course
Please have a look through the report before trying to figure out the code.

# Summary 
For this research project, several one-, two-, and three-dimensional numerical simulations for solving heat transfer and fluid dynamics problems have been created and evaluated. The first simulations were made using Wolfram Mathematica and focused on heat conduction problems within simple geometric objects, such as squares, cubes, circles, and slabs, which can be solved numerically. Steady-state problems, with and without heat production involved, as well as unsteady-state problems, were solved with these simulations. The remaining simulations were made using SolidWorks and include real-life applications of fluid dynamics and heat transfer via conduction, convection, and radiation. An analytical approach for solving steady and unsteady states has been presented. The results of different examples have been evaluated using Solidworks tools, with a focus on the results for transient simulation.
